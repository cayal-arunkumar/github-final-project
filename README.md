# Simple Interest Calculator

A simple, lightweight tool to calculate simple interest.

## Formula
Simple Interest (SI) = (Principal × Rate × Time) / 100

## Inputs
- Principal (P): the initial amount of money
- Rate (R): annual interest rate (%)
- Time (T): time period in years

## Usage
Run the script and enter the principal, rate, and time when prompted.
The tool outputs the calculated simple interest.

## Example
P = 1000, R = 5, T = 2
SI = (1000 × 5 × 2) / 100 = 100
